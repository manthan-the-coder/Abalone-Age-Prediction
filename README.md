# 🐚 Abalone Age Prediction App
**Live Demo:** [https://ml-wonder-web.lovable.app](https://ml-wonder-web.lovable.app)

## 📌 Overview
This project predicts the age of an Abalone using physical measurements. It consists of a **Random Forest Regression** model and a custom-built **React frontend**.

## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS (Built via Lovable)
- **Machine Learning:** Python, Scikit-Learn, Pandas
- **Deployment:** Lovable (UI) & GitHub (Source Code)
